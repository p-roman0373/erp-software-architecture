# 2. Architecture Constraints

## Decisiones Tecnológicas

- Backend desarrollado en Java utilizando Spring Boot.
- Frontend desarrollado como Single Page Application (SPA) con React.
- Base de datos relacional PostgreSQL.
- Comunicación entre frontend y backend mediante HTTPS y formato JSON.
- Uso de PlantUML para diagramación.
- Documentación estructurada con plantilla arc42.
- Control de versiones mediante repositorio público en GitHub.
