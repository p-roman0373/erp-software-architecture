# 5. Building Block View

## Descripción de Contenedores

El sistema ERP está compuesto por los siguientes contenedores principales:

### SPA Web (React)
Interfaz de usuario que permite a los administradores gestionar productos, proveedores y órdenes de compra.

### API Monolítica (Spring Boot)
Contiene la lógica de negocio, validaciones y procesamiento de datos.

### Base de Datos (PostgreSQL)
Almacena la información estructurada del sistema, incluyendo productos, proveedores y órdenes de compra.
