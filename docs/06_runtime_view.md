# 6. Runtime View

## Escenario: Registrar un Producto

Este escenario describe el flujo cuando un administrador registra un nuevo producto en el sistema.

### Flujo del Proceso

1. El administrador completa el formulario en la interfaz web.
2. La SPA envía una solicitud POST a la API.
3. La API valida los datos recibidos.
4. Si los datos son correctos, se almacenan en la base de datos.
5. La API responde con confirmación de creación.
6. La interfaz muestra mensaje de éxito al usuario.
