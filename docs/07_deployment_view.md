El sistema puede desplegarse en un servidor en la nube.

## Arquitectura de Despliegue Propuesta

- Servidor de Aplicación: ejecuta la API desarrollada en Spring Boot.
- Servidor Web: entrega la aplicación React.
- Servidor de Base de Datos: aloja PostgreSQL.
- Comunicación protegida mediante protocolo HTTPS.

Esta arquitectura permite escalabilidad y separación de responsabilidades.
